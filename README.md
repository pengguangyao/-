# react-admin
基于react，ant-design，dva ，mongodb后台管理系统

## 项目概述
 
此项目就是实现一个有管理权限，用来管理公司相关信息的一个平台。

## 特性

- 实现登录注册功能
- 基于Antd UI 设计语言，提供后台管理系统常见使用场景
- 浅度响应式设计
- webpack打包处理路由时，实现Javascript模块化按需动态dynamic加载
- 基于Tab面板的导航栏切换
